# Portfolio - Coming Soon Page

## Project Type
Simple single-file HTML website (no build tools required)

## Files
- `index.html` - Main coming soon page

## Key Details
- Domain: jameer.online
- Email capture: Formspree (free tier)
- Analytics: Google Analytics 4 (GA4)

## Dependencies
- None (pure HTML/CSS/JS)

## Testing
- Open `index.html` directly in browser
- Test email form submission
- Verify GA4 tracking fires (check GA4 DebugView)

## External Services
- **Formspree**: Sign up at formspree.io, create form, replace `action` URL in index.html
- **Google Analytics 4**: Create property at analytics.google.com, get measurement ID (G-XXXXXXX), replace in tracking script
